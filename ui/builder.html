Maestro Builder

Create — drag nodes from the palette. Every node's properties live in the right panel; the Instruction box is all an agent step needs (pin a skill under Advanced for reproducible runs).

Connect — drag from a node's ⚪ output port onto another node. Arrows back to an earlier node create a loop: the engine re-runs from there and a repeat-limit (max visits) keeps it bounded — click the dashed edge to tune it.

Gates — each option is its own outgoing edge. Edit options in the inspector.

Parallel / subworkflow — double-click (or use the card's open buttons) to drill into branches; breadcrumbs bring you back.

Export — one click, always validated. Positions are saved in a ui: key the engine ignores. YAML comments are not preserved on export — keep prose in description and node notes.

Run a workflow with /maestro <slug> <file> in Claude Code / Cursor. Validate anytime with python3 engine/maestroctl.py validate <file>.
